# Week 5 Day 26 Assignment

This repository contains my submission for the Week 5 Day 26 assignment.

## About the assignment

In this assignment, I worked on a case study based on Pharmalink, a B2B medicine distribution platform. The task was related to machine learning concepts such as problem identification, ML pipeline stages, evaluation strategy, and handling different customer behavior by city tier.

## Files in this repository

- `answers.md` - contains my answers to the assignment questions
- `README.md` - contains a short overview of this repository

## What I learned

From this assignment, I learned how to:
- identify whether a problem is supervised or unsupervised
- understand the difference between classification and regression
- connect business problems with machine learning pipeline stages
- choose between precision and recall based on business needs
- think about different customer behavior patterns in real-world datasets
